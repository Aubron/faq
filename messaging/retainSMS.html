How long does Bandwidth retain messages?

  1. FAQ
  2. Messaging
  3. How long does Bandwidth retain messages?

Bandwidth keeps messages sent with the v1 api for 30days. After 30 days, you will no longer be able to fetch that particular message.

For information on how to list messages, see the list messages and for how to get more information on a specific message see the message inforesource
